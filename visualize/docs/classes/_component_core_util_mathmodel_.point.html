Class Point<T, U>

Point:

  • Single point consists of pair of x and y with its own name.
  • It has polymorphic type T and U.
  • example:

    type T for number type and type U for string (x, y) called this point as "x_1"

Type parameters

  • T

    the point's value type. Default type is number.

  • U

    the point's name type. Default type is string.
