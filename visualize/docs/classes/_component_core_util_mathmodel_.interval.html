Class Interval<V, T, U>

Interval:

  • Array of points with name.
  • It has polymorphic type T and U same as point class.
  • example:

    type T for number type and type U for string [(x, y), (x_1, y_1) ... ] called this interval as "interval_1"

Type parameters

  • V

    the array's name type. Default type is string.

  • T

    the array's point value type. Default type is number.

  • U

    the array's point name type. Default type is string.
